# Evaluación del Modelo

## Metodología

Evaluación mediante:
- train / validation / test
- métricas estándar

---

## Métricas

- Accuracy
- Precision
- Recall
- F1-score

---

## Interpretación

Una predicción positiva indica:

El perfil es similar a usuarios que reportaron mejora.

No garantiza mejora real.

---

## Limitaciones

- Datos auto-reportados
- Tamaño de muestra reducido
- Posibles sesgos
